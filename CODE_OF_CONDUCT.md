# Contributor Covenant 行为准则

## 我们的承诺

为了营造一个开放、友好的环境，我们作为贡献者和维护者承诺：无论年龄、体型、残疾、种族、性别特征、性别认同和表达、经验水平、教育背景、社会 status、是否残疾、个人外貌、种族、宗教或性取向，参与本项目的人都不会受到骚扰。

## 我们的标准

有助于创造积极环境的行为包括但不限于：

* 使用欢迎和包容的语言
* 尊重不同的观点和经验
* 优雅地接受建设性的批评
* 关注对社区最有利的事情
* 对其他社区成员表示同理心

不可接受的行为包括但不限于：

* 使用性暗示的语言或图像，以及不受欢迎的性关注或诱惑
* 挑衅、侮辱或贬低性评论，人身攻击或政治攻击
* 公开或私下骚扰
* 未经明确许可，发布他人的私人信息，如物理地址或电子地址
* 其他在专业环境中可能被合理认为不适当的行为

## 我们的责任

项目维护者有责任澄清可接受行为的标准，并应采取适当和公平的纠正措施来应对任何不可接受的行为。

项目维护者有权删除、编辑或拒绝与本行为准则不一致的评论、提交、代码、wiki 编辑、问题和其他贡献。他们有权暂时或永久禁止任何他们认为不合适、威胁、冒犯或有害的贡献者。

## 范围

当个人代表项目或其在公共领域中的社区时，本行为准则适用于项目空间和公共空间。代表项目或社区的例子包括但不限于使用官方项目电子邮件地址、通过官方社交媒体账户发布或在线或离线活动中担任指定或提名代表。

## 执行

虐待、骚扰或其他不可接受的行为可以通过联系项目维护者 [2240678683@qq.com] 进行报告。所有投诉都将得到审查和调查，并将根据具体情况做出回应。项目维护者有义务对事件的报告者保密。

项目维护者将遵循这些准则，以确定对报告的事件采取何种行动是适当的：

### 1. 纠正

**社区影响**：使用不适当的语言或其他被视为不专业或不欢迎的行为。

**后果**：向受害者私下书面警告，并提供清晰的解释说明问题所在。可能会要求公开道歉。

### 2. 警告

**社区影响**：单次事件或一系列行为。

**后果**：警告，并提供清晰的解释说明问题所在。不得进行进一步的不良行为。不得与涉及此行为的人员进行互动，包括主动请求或被动地避免回应。

### 3. 临时禁止

**社区影响**：严重的违规行为，包括持续的不良行为。

**后果**：临时禁止与项目或其社区的任何人进行任何形式的互动，为期指定的一段时间。在此期间，不得与涉及此行为的人员进行互动，包括主动请求或被动地避免回应。违反这些条款可能会导致永久禁止。

### 4. 永久禁止

**社区影响**：表现出严重违反社区标准，包括持续的不良行为、骚扰个人或攻击或贬低一类个人。

**后果**：永久禁止在项目或其社区内进行任何形式的公开互动。
