## 安装命令

```bash
# 基础安装
pip install requests urllib3 PyYAML python-dateutil apscheduler

# 或使用本文件
pip install -r requirements.txt
```

## 可选依赖

### 如果使用WechatSogou（微信公众号爬虫）
```bash
cd D:\hfut_info_monitor
git clone https://github.com/Chyroc/WechatSogou.git
cd WechatSogou
pip install -r requirements.txt
```

### 如果使用go-cqhttp（QQ监控）
```bash
cd D:\hfut_info_monitor
wget https://github.com/Mrs4s/go-cqhttp/releases/download/v1.2.0/go-cqhttp_windows_amd64.exe
mv go-cqhttp_windows_amd64.exe go-cqhttp.exe
```

### 如果使用Ollama（本地AI分析）
```bash
# 安装Ollama
# 下载：https://ollama.com/download

# 拉取模型
ollama pull qwen2:7b
```
